# Random-Password-Generator

Хафизова Элина

Данное приложение представляет собой графический генератор случайных паролей с возможностью настройки параметров генерации, сохранения истории созданных паролей и экспорта данных в формат JSON.
Программа разработана с использованием стандартной библиотеки Python tkinter для создания пользовательского интерфейса и библиотеки random для генерации случайных последовательностей символов.
Примеры использования
Пример 1: Генерация простого пароля (только цифры)
Цель: Создать PIN-код из 6 цифр.
Шаги:
Запустить приложение
Установить ползунок длины на значение 6
Снять галочки с пунктов "A-Z", "a-z" и "!@#$%^&*"
Оставить включённым только пункт "0-9"
Нажать кнопку "GENERATE PASSWORD"
Ожидаемый результат: Пароль из 6 случайных цифр, например: 739284
Проверка: В истории появится запись с текущим временем, длиной 6 и маскированным паролем 739***284.
Пример 2: Создание сложного пароля для аккаунта
Цель: Сгенерировать надёжный пароль длиной 16 символов со всеми типами символов.
Шаги:
Установить длину пароля: 16
Убедиться, что все чекбоксы включены (A-Z, a-z, 0-9, спецсимволы)
Нажать "GENERATE PASSWORD"
Ожидаемый результат: Пароль типа K9#mP2$vL5@nQ8*w
Проверка:
Пароль содержит заглавные буквы (K, P, L, Q)
Пароль содержит строчные буквы (m, v, n, w)
Пароль содержит цифры (9, 2, 5, 8)
Пароль содержит спецсимволы (#, $, @, *)
Пример 3: Проверка валидации минимальной длины
Цель: Убедиться, что программа не позволяет создать слишком короткий пароль.
Шаги:
Попытаться установить ползунок на значение меньше 4 (или вручную изменить код)
Нажать "GENERATE PASSWORD"
Ожидаемый результат: Появится окно ошибки с текстом: "Minimum password length is 4 characters."
Пример 4: Проверка валидации отсутствия символов
Цель: Проверить реакцию программы, если не выбран ни один тип символов.
Шаги:
Снять все галочки с чекбоксов
Нажать "GENERATE PASSWORD"
Ожидаемый результат: Появится окно ошибки: "Please select at least one character type..."
Пример 5: Работа с историей
Цель: Проверить сохранение и загрузку истории.
Шаги:
Сгенерировать 3 разных пароля с разными настройками
Закрыть приложение
Снова открыть приложение
Ожидаемый результат: В таблице истории отобразятся все 3 ранее сгенерированных пароля с их параметрами.
